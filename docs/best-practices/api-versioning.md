# API Versioning Policy

PhilJS follows Semantic Versioning (SemVer) 2.0.0.

## Breaking Changes

- Occur only in MAJOR versions (e.g., 1.x -> 2.x).
- Deprecations are announced at least one MINOR version ahead.
- We provide `codemods` for most breaking changes.

## Minor Versions

- New features, non-breaking.
- Backward compatible.

## Patch Versions

- Bug fixes only.
