# Changelog

All notable changes to PhilJS are documented here. We follow Semantic Versioning.

## [0.1.0] - 2025-12-15

Status: Current preview

Highlights:
- TypeScript-first core primitives (signals, memos, effects)
- JSX runtime and rendering pipeline
- Islands architecture and SSR primitives
- Docs site and example apps aligned to @philjs/*

Breaking changes:
- N/A (initial preview)

Dependencies:
- Node.js 24+ (Node 25 supported)
- TypeScript 6.x

## [0.0.1] - 2025-09-01

Status: Initial scaffold

Highlights:
- Monorepo structure and package layout
- Initial tooling and examples

## Versioning Policy

PhilJS is pre-1.0.0. Minor versions may include breaking changes with clear migration notes.

- Major (1.0.0 and above): Breaking changes with migration guide
- Minor (0.X.0): Features and possible breaking changes (documented)
- Patch (0.X.Y): Bug fixes and small improvements

---

## Support Timeline

- Current preview (0.1.x): Active development and support
- Older previews: Best-effort fixes only
