# SEO Optimization

## Meta Tags

Use the `<Meta>` component (compatible with `react-helmet`) to manage head tags.

```tsx
<Meta>
  <title>My Page - PhilJS</title>
  <meta name="description" content="Best page ever" />
</Meta>
```

## Sitemap

PhilJS automatically generates `sitemap.xml` for static routes during build if enabled in `philjs.config.ts`.
