
# Backend Adapters (`@philjs/backend-adapters`)

Unified integration layer for server-side protocols.

## Features
- **Inertia.js**: Protocol adapter for modern monoliths.
- **Phoenix Channels**: WebSocket client for Elixir Phoenix.
- **ActiveRecord**: Chainable generic ORM wrapper.

```typescript
import { InertiaPage } from '@philjs/backend-adapters';
```
