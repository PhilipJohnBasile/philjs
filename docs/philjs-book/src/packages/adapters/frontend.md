
# Frontend Adapters (`@philjs/frontend-adapters`)

State management and frontend logic.

## Supported Patterns
- **MobX**: Observables and reactions.
- **Redux**: Actions, reducers, and DevTools middleware.
- **Recoil**: Atom families and selectors.
- **XState**: Finite state machines.

```typescript
import { useObservable } from '@philjs/frontend-adapters';
```
