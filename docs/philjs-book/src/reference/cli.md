# CLI Reference

Use `philjs-cli` for application development.

## Install

```bash
pnpm add -D philjs-cli
```

## Common commands

```bash
philjs dev
philjs build
philjs preview
philjs test --coverage
philjs generate component Button
philjs generate route dashboard
philjs generate-types
```

## Notes

PhilJS is TypeScript-only. Do not use JavaScript generation flags or JS-only entry points.
