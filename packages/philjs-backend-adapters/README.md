
# @philjs/backend-adapters

Unified Backend Integration Layer.

## Features
- **Inertia.js**: Server-side rendering protocol adapter.
- **Phoenix Channels**: WebSocket client for Elixir Phoenix.
- **ActiveRecord**: Chainable ORM query simulation.

## Usage
```typescript
import { InertiaPage } from '@philjs/backend-adapters';
const page = new InertiaPage();
```
