
# @philjs/crypto

Post-Quantum Cryptography.

## Features
- **Quantum Resistant**: Lattice-based algorithms (Kyber, Dilithium).
- **Future Proof**: Protects data against Shor's Algorithm attacks.

## Usage
```typescript
import { PQC } from '@philjs/crypto';
const cipher = PQC.encrypt(data, pubKey);
```
