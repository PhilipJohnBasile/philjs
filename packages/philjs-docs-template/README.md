# @philjs/docs-template

VitePress-like documentation template for PhilJS projects

Part of the **PhilJS** ecosystem.

## Installation

```bash
npm install @philjs/docs-template
# or
pnpm add @philjs/docs-template
```

## Features

- **Production Ready**: Fully typed and optimized.
- **Integrated**: Designed to work seamlessly with @philjs/core.
- **Modern**: Built for ESNext and Node.js 24+.
