
# @philjs/frontend-adapters

State Management and Frontend Logic.

## Features
- **MobX**: Observable state primitives.
- **Redux**: Middleware and DevTools simulation.
- **Recoil**: Atoms and selectors family support.
- **XState**: Finite state machine hooks.

## Usage
```typescript
import { useObservable } from '@philjs/frontend-adapters';
const state = useObservable({ count: 0 });
```
