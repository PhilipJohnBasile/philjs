
# @philjs/integration

Third-Party Integrations & SDKs.

## Features
- **Algolia**: Search/Indexing logic simulation.
- **Web3**: Wagmi/Viem hook mocks (`useAccount`).
- **CMS**: Strapi and Clerk adapter simulations.

## Usage
```typescript
import { search } from '@philjs/integration';
const results = await search('query');
```
