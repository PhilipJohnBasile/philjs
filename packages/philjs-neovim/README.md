# @philjs/neovim

Neovim adapter helper

Part of the **PhilJS** ecosystem.

## Installation

```bash
npm install @philjs/neovim
# or
pnpm add @philjs/neovim
```

## Features

- **Production Ready**: Fully typed and optimized.
- **Integrated**: Designed to work seamlessly with @philjs/core.
- **Modern**: Built for ESNext and Node.js 24+.
