# PhilJS RPC Treaty Client Examples

This directory contains comprehensive examples demonstrating the Treaty client implementation.

## Examples

### 1. Basic Usage (treaty-basic.ts)

Shows fundamental treaty patterns including GET, POST, PUT, DELETE operations with full type safety.

### 2. Authentication (treaty-auth.ts)

Demonstrates JWT authentication, protected routes, role-based access, and token management.

### 3. Advanced Features (treaty-advanced.ts)

Shows file uploads, retry logic, timeouts, batch requests, pagination, and WebSocket support.

## Running Examples

```bash
npx tsx examples/treaty-basic.ts
```

See individual files for detailed documentation.
