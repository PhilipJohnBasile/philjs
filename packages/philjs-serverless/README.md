# @philjs/serverless

Unified serverless adapter for AWS Lambda, Vercel, Cloudflare Workers, and more

Part of the **PhilJS** ecosystem.

## Installation

```bash
npm install @philjs/serverless
# or
pnpm add @philjs/serverless
```

## Features

- **Production Ready**: Fully typed and optimized.
- **Integrated**: Designed to work seamlessly with @philjs/core.
- **Modern**: Built for ESNext and Node.js 24+.
