
# @philjs/social

Decentralized Social Web (Fediverse).

## Features
- **ActivityPub**: Native implementation of W3C generic social protocol.
- **Federation**: Connect with Mastodon, PixelFed, and Lemmy.

## Usage
```typescript
import { ActivityPub } from '@philjs/social';
ActivityPub.publish('@phil', 'Hello Fediverse!');
```
