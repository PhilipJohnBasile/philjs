# @philjs/tailwind-ui

Tailwind Ui integration for PhilJS. Robust, type-safe, and production-ready.

Part of the **PhilJS** ecosystem.

## Installation

```bash
npm install @philjs/tailwind-ui
# or
pnpm add @philjs/tailwind-ui
```

## Features

- **Production Ready**: Fully typed and optimized.
- **Integrated**: Designed to work seamlessly with @philjs/core.
- **Modern**: Built for ESNext and Node.js 24+.
