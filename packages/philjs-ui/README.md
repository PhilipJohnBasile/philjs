
# @philjs/ui

Advanced UI Component Library.

## Features
- **Smart Components**: `Checkout`, `VideoRoom`, `GeoJSONMap`.
- **Visualization**: `Chart` (Recharts-like), `RemotionPlayer`.
- **Animation**: `GSAP` timeline simulation hooks.

## Usage
```tsx
import { Checkout } from '@philjs/ui';
<Checkout amount={5000} currency="usd" />
```
